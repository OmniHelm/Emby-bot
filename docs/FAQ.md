## 👤 账户相关

### Q1: 如何创建 Emby 账户？

**A:**

**步骤：**
1. 点击 `/start` 召唤主面板
2. 点击 **🎨 创建账户** 按钮
3. 按格式输入：`用户名 安全码`
   - 示例：`张三 1234`
4. 等待系统创建（约 5-10 秒）
5. 记录用户名和密码

**注意：**
- 用户名支持中英文，不支持特殊字符
- 安全码必须是 4-6 位数字
- 密码仅显示一次，请妥善保管

---

### Q2: 忘记密码怎么办？

**A:**

**方法一：重置密码**
1. 在主面板点击 **👤 用户功能**
2. 点击 **🔑 重置密码**
3. 输入安全码验证
4. 系统会生成新密码

**方法二：联系管理员**
- 如果忘记安全码，只能联系管理员
- 提供你的 Telegram ID
- 验证身份后由管理员重置

**安全提示：**
- 重置后旧密码立即失效
- 建议登录后在客户端中修改
- 不要频繁重置密码

---

### Q3: 可以修改用户名吗？

**A:**

**不可以。**

**原因：**
- 用户名创建后无法修改
- 这是 Emby 服务器的限制
- 修改会导致数据丢失

**解决方案：**
- 只能删除旧账户
- 重新创建新账户
- 但旧数据无法恢复

**建议：**
- 创建前想好用户名
- 选择容易记住的名字
- 避免使用敏感信息

---

### Q4: 忘记安全码怎么办？

**A:**

**安全码是敏感操作的验证密码，忘记后只能联系管理员。**

**联系管理员：**
1. 在群组中 @管理员
2. 说明你的情况
3. 提供 Telegram ID
4. 验证身份

**验证方式：**
- 回答注册时的问题
- 提供账户创建时间
- 其他身份验证

**安全建议：**
- 安全码要容易记住
- 不要使用生日等易被猜到的数字
- 可以记录在安全的地方

---

### Q5: 如何删除账户？

**A:**

**删除步骤：**
1. 主面板 → **👤 用户功能**
2. 点击 **🗑️ 删除账号**
3. 输入安全码确认
4. 再次确认删除

**警告：**
- 🔴 此操作无法撤销
- 🔴 所有数据将永久删除
- 🔴 收藏和观看记录全部丢失
- 🔴 剩余时长作废

**删除后：**
- 可以重新创建账户
- 但需要重新获取注册资格
- 旧数据无法恢复

## 📺 使用相关

### Q6: 可以在多个设备上使用吗？

**A:**

**可以，但有限制。**

**设备数量：**
- ✅ 可以在多个设备上登录
- ✅ 没有登录设备数量限制
- ❌ 同时播放设备数限制为 2 台（默认）

**什么是同时播放？**
- 指同一时间正在播放视频的设备数
- 暂停或停止后不计入
- 仅浏览不计入

**超出限制时：**
- 会提示"已达到播放设备上限"
- 需要停止其他设备的播放
- 或等待其他设备自动停止

**查看设备：**
1. 主面板 → **👤 用户功能**
2. 点击 **💠 我的设备**
3. 查看所有登录设备
4. 可以删除不需要的设备

---

### Q7: 为什么无法播放某些视频？

**A:**

**可能的原因：**

**1. 编解码问题**
- 设备不支持该视频编码
- 解决：开启转码播放

**2. 网络问题**
- 网速太慢
- 解决：降低画质或更换网络

**3. 文件损坏**
- 视频文件本身有问题
- 解决：更换其他视频或报告管理员

**4. 权限问题**
- 媒体库被隐藏
- 解决：在 **🎬 显示/隐藏** 中启用

**5. 设备性能**
- 设备性能不足
- 解决：使用性能更好的设备

**通用解决方法：**
1. 尝试其他视频，确认是个别问题还是全部
2. 切换画质到较低档位
3. 开启转码播放
4. 更换网络（WiFi/4G/5G）
5. 重启 Emby 客户端
6. 联系管理员报告问题

### Q9: 如何使用收藏功能？

**A:**

**添加收藏：**
1. 在 Emby 客户端中找到影片
2. 点击"收藏"或"❤️"图标
3. 自动添加到收藏列表

**查看收藏：**

**方法一：机器人**
1. 主面板 → **👤 用户功能**
2. 点击 **💖 我的收藏**
3. 浏览收藏列表

**方法二：客户端**
1. 打开 Emby 客户端
2. 找到"收藏"或"我的列表"
3. 查看所有收藏

**取消收藏：**
1. 找到已收藏的影片
2. 再次点击"收藏"图标
3. 自动从收藏中移除

**优点：**
- 跨设备同步
- 快速找到喜欢的内容
- 追踪更新状态

### Q10: 如何切换服务器线路？

**A:**

**查看可用线路：**
1. 主面板点击 **🌐 服务器**
2. 查看所有可用线路
3. 记录服务器地址

**更换线路：**
1. 在 Emby 客户端中
2. 删除当前服务器
3. 添加新的服务器地址
4. 重新登录

**如何选择线路：**
- 选择延迟较低的
- 选择下载速度快的
- 不同时段可能不同

**测试方法：**
1. 逐个测试各条线路
2. 播放视频观察流畅度
3. 选择最流畅的线路

## 🎟️ 续期相关

### Q11: 如何续期账户？

**A:**

**方法一：使用续期码**
1. 获取续期码（购买或活动）
2. 发送：`/exchange 续期码`
3. 系统自动续期

**方法二：兑换商店**
1. 点击 **🏪 兑换商店**
2. 查看可用续期码
3. 使用积分兑换
4. 自动应用到账户

**方法三：联系管理员**
- 在群组联系管理员
- 购买续期服务
- 管理员手动续期

---

### Q12: 账户到期后会怎样？

**A:**

**到期后：**
- ❌ 无法登录 Emby 服务器
- ❌ 已登录的设备会被踢出
- ❌ 无法播放任何视频

**数据保留：**
- ✅ 收藏记录保留 7 天
- ✅ 观看进度保留 7 天
- ✅ 账户信息保留 7 天

**7 天内续期：**
- 所有数据恢复
- 继续使用服务

**超过 7 天：**
- 数据可能被清理
- 需要联系管理员确认

**如何避免：**
- 设置到期提醒
- 提前续期
- 保持积分余额

---

### Q13: 续期码可以叠加使用吗？

**A:**

**可以叠加。**

**叠加规则：**
- 多个续期码的天数会累加
- 在当前到期时间基础上增加
- 没有上限限制

**示例：**
- 当前到期时间：2024-12-01
- 使用 30 天续期码
- 新到期时间：2024-12-31
- 再使用 30 天续期码
- 最终到期时间：2025-01-30

**注意：**
- 已过期账户从当前时间开始计算
- 未过期账户从原到期时间延长
- 续期码使用后立即生效

---

### Q14: 续期码有效期多久？

**A:**

**取决于续期码类型：**

**永久有效码：**
- 没有使用期限
- 随时可以使用
- 不会过期

**限时有效码：**
- 有明确的使用期限
- 过期后无法使用
- 兑换时会显示有效期

**如何查看：**
- 兑换商店会显示有效期
- 管理员提供码时会说明
- 使用时系统会检查

**建议：**
- 获得后尽快使用
- 不要囤积太多
- 接近到期优先使用

### Q17: 视频卡顿怎么办？

**A:**

**诊断步骤：**

**1. 检查网络**
```
测试方法：
- 测速网站测试速度
- 至少需要 10Mbps
- WiFi 优于移动网络
```

**2. 降低画质**
```
设置方法：
- 打开播放设置
- 选择较低码率
- 推荐：720P 或 480P
```

**3. 开启转码**
```
设置方法：
- 客户端设置
- 开启硬件加速
- 让服务器转码
```

**4. 更换线路**
```
操作方法：
- 点击 🌐 服务器
- 查看可用线路
- 选择延迟最低的
```

**5. 更换设备**
```
建议：
- 使用性能更好的设备
- 确保系统版本较新
- 清理设备存储空间
```

**永久解决：**
- 升级网络带宽
- 更换更好的设备
- 使用有线连接

**方法三：转码**
1. 开启服务器转码
2. 自动转换字幕编码
3. 解决乱码问题

**报告问题：**
- 告诉管理员哪个视频
- 说明字幕问题
- 管理员会修复

### Q20: 如何设置播放质量？

**A:**

**在客户端设置：**

**移动端：**
1. 打开 Emby 应用
2. 设置 → 播放
3. 设置以下选项：
   - 最大码率
   - 网络质量
   - 本地媒体质量

**网页版：**
1. 播放视频时
2. 点击设置图标
3. 选择画质
4. 立即生效

**推荐设置：**

**家庭 WiFi（50Mbps+）：**
- 原画质或 1080P
- 开启硬件加速

**移动网络（4G）：**
- 720P
- 开启自适应码率

**慢速网络：**
- 480P 或更低
- 开启转码
