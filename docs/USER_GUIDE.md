## 🚀 快速开始

### 第一步：加入群组

使用 EmbyBot 前，你需要先加入官方群组/频道：

1. 点击机器人发送的群组链接
2. 加入所有必需的群组/频道
3. 返回机器人对话

### 第二步：开始使用

发送 `/start` 命令启动机器人：

```
/start
```

系统会自动将你录入数据库，并显示主面板。

### 第三步：创建账户

在主面板中：

1. 点击 **🎨 创建账户** 按钮
2. 按提示输入：`用户名 安全码`
   - 示例：`张三 1234`
3. 等待系统创建账户
4. 记录你的用户名和密码

### 第四步：登录 Emby

1. 下载 Emby 客户端
2. 输入服务器地址
3. 使用创建的账户登录
4. 开始享受影音服务

## 🎯 常用操作

### 1. 创建 Emby 账户

**步骤：**

1. 点击 `/start` 打开主面板
2. 点击 **🎨 创建账户**
3. 输入格式：`用户名 安全码`
   ```
   示例：张三 1234
   ```
4. 等待系统处理（约5-10秒）
5. 记录账户信息

**规则说明：**

**用户名：**
- ✅ 支持中文、英文、数字
- ✅ 可以使用下划线
- ❌ 不支持特殊字符（@#$%^&*）
- 📏 长度：2-20 个字符

**安全码：**
- ✅ 必须是纯数字
- 📏 长度：4-6 位
- 💡 建议使用容易记住的数字
- 🔐 用于敏感操作验证

**注意事项：**
- ⚠️ 密码仅显示一次，请妥善保管
- ⚠️ 用户名创建后无法修改
- ⚠️ 安全码请勿告诉他人

### 2. 重置密码

**适用场景：**
- 忘记密码
- 密码泄露
- 定期更换密码

**操作步骤：**

1. 点击 `/start` 打开主面板
2. 点击 **👤 用户功能**
3. 点击 **🔑 重置密码**
4. 输入安全码验证
5. 获取新密码

**注意：**
- 重置后旧密码立即失效
- 新密码会立即发送给你
- 建议登录后在 Emby 客户端中修改

### 3. 续期账户

**方式一：使用续期码**

1. 获取续期码（购买或活动获得）
2. 发送命令：`/exchange 续期码`
3. 等待系统处理
4. 查看新的到期时间

**方式二：兑换商店**

1. 点击 **🏪 兑换商店**
2. 查看可用的续期码
3. 使用积分兑换
4. 自动续期

**注意：**
- 续期码只能使用一次
- 可以在到期前或到期后使用
- 多个续期码的天数会累加

### 4. 查看收藏

**功能说明：**
查看你在 Emby 中收藏的影片列表。

**操作步骤：**

1. 点击 **👤 用户功能**
2. 点击 **💖 我的收藏**
3. 浏览收藏列表
4. 点击影片可查看详情

### 5. 管理设备

**功能说明：**
查看和管理登录你账户的设备。

**操作步骤：**

1. 点击 **👤 用户功能**
2. 点击 **💠 我的设备**
3. 查看设备列表
4. 可以删除不认识的设备

**安全提示：**
- 定期检查设备列表
- 发现异常设备立即删除
- 删除后修改密码

### 6. 显示/隐藏媒体库

**功能说明：**
自定义显示或隐藏特定的媒体库。

**操作步骤：**

1. 点击 **👤 用户功能**
2. 点击 **🎬 显示/隐藏**
3. 选择要操作的媒体库
4. 切换显示状态

**注意：**
- 隐藏后在客户端中不可见
- 随时可以重新显示
- 不影响收藏和观看记录

## ❓ 常见问题

### Q: 如何登录 Emby？

**A:**

1. 下载 Emby 客户端
   - iOS: App Store 搜索 "Emby"
   - Android: Google Play 或第三方应用商店
   - 电脑: 访问 emby.media 下载
   - 网页: 直接访问服务器地址

2. 添加服务器
   - 输入服务器地址（机器人会提供）
   - 点击"连接"

3. 登录账户
   - 输入用户名
   - 输入密码
   - 点击"登录"

### Q: 忘记密码怎么办？

**A:**

1. 点击 `/start` 打开主面板
2. 点击 **👤 用户功能**
3. 点击 **🔑 重置密码**
4. 输入安全码验证
5. 获取新密码

**如果忘记安全码：**
- 联系管理员协助重置
- 提供你的 Telegram ID
- 验证身份后重置

### Q: 账户到期后会怎样？

**A:**

**立即影响：**
- ❌ 无法登录 Emby 服务器
- ❌ 已登录的设备会被踢出

**数据保留：**
- ✅ 收藏记录保留 7 天
- ✅ 观看进度保留 7 天
- ✅ 7 天内续期可恢复

**如何续期：**
1. 使用续期码：`/exchange 续期码`
2. 购买续期码：点击 **🏪 兑换商店**
3. 联系管理员

### Q: 可以在多个设备上使用吗？

**A:**

可以，但有限制：

**设备数量：**
- 可以在多个设备上登录
- 同时播放设备数有限制（默认2台）

**如何查看设备：**
1. 点击 **💠 我的设备**
2. 查看所有登录设备
3. 可以删除不需要的设备

**超出限制时：**
- 会提示"已达到播放设备上限"
- 停止其他设备的播放即可
- 或删除不用的设备

### Q: 如何邀请好友？

**A:**

**获取邀请码：**
1. 点击 **🏪 兑换商店**
2. 使用积分兑换注册码
3. 分享给好友

**好友注册：**
1. 好友点击你的邀请链接
2. 或使用 `/exchange 注册码`
3. 创建账户

**邀请奖励：**
- 好友成功注册后
- 你会获得积分奖励
- 可用于兑换续期码

### Q: 视频卡顿怎么办？

**A:**

**原因分析：**
- 网络速度慢
- 设备性能不足
- 视频码率过高

**解决方法：**

1. **降低画质**
   - 在播放设置中
   - 选择较低的码率
   - 如 720P 或 480P

2. **使用转码**
   - 开启硬件加速
   - 让服务器转码

3. **更换线路**
   - 点击 **🌐 服务器**
   - 查看可用线路
   - 选择延迟较低的

4. **检查网络**
   - 切换到 WiFi
   - 重启路由器
   - 联系网络运营商

### Q: 如何删除账户？

**A:**

**删除步骤：**

1. 点击 **👤 用户功能**
2. 点击 **🗑️ 删除账号**
3. 确认删除操作
4. 账户立即删除

**删除后果：**
- 🔴 无法撤销
- 🔴 所有数据永久删除
- 🔴 收藏和观看记录丢失
- 🔴 剩余时长作废

**重新注册：**
- 删除后可以重新创建
- 但需要重新获取注册资格
- 旧数据无法恢复
